设A，B为矩阵，已知两个线性子空间 $V_1, V_2 \subseteq \mathbb{R}^n$（或 $\mathbb{C}^n$），且$V_{1}和V_{2}$分别是齐次方程组$Ax=0,Bx=0$的解空间，求：  $\dim(V_1 \cap V_2)$

$$dim(V_{1}+V_{2})=dim(V_{1})+dim(V_{2})-dim(V_{1} \cap V_{2})$$

因此：
$$dim(V_{1} \cap V_{2})=dim(V_{1})+dim(V_{2})-dim(V_{1}+V_{2})$$

**步骤**：

1. 分别求 $\dim V_1, \dim V_2$
	先找A,B的秩，dim=列数-rank
2. 求 $\dim(V_1 + V_2)$
	$V_{1} \cap V_{2} = \{ x \mid Ax = 0 \text{ 且 } Bx = 0 \} = \left\{ x \mid \begin{pmatrix} A \\ B \end{pmatrix} x = 0 \right\}$
	求$dim V_{1} \cap V_{2}$就是令$C=\begin{pmatrix}A\\B\end{pmatrix}$
	然后求rank（C）,dim=列数-rank（C）

# 为什么当 rank(A) ≤ n < m 时，齐次方程组必有非零解
[[100 学习相关/111 研一课程/矩阵理论/考试用的一些/线性映射与矩阵/矩阵的秩|矩阵的秩]]

## 🔍 核心结论
对于 $m \times n$ 矩阵 $A$，如果满足：
$$\text{rank}(A) \leq n < m$$
那么齐次线性方程组 $A\mathbf{x} = \mathbf{0}$ 必定存在**非零解**。

## 📐 三个理解角度

### 1. 零空间维度视角
齐次方程组的解空间（零空间）维度为：
$$\dim(\text{Null}(A)) = n - \text{rank}(A)$$

由于 $\text{rank}(A) \leq n$，可得：
$$\dim(\text{Null}(A)) \geq 0$$

**关键点**：当 $\text{rank}(A) < n$ 时，
$$\dim(\text{Null}(A)) = n - \text{rank}(A) > 0$$
即解空间至少是1维的，存在无穷多个非零解。

### 2. 列向量线性相关性视角
矩阵 $A$ 包含 $n$ 个 $m$ 维列向量。

由于 $n < m$（向量个数小于维度），但：
- 最大可能秩为 $\min(m, n) = n$
- 已知 $\text{rank}(A) \leq n$，包含两种情况：

**当 $\text{rank}(A) < n$ 时**，列向量线性相关，存在不全为零的系数 $x_1, ..., x_n$ 使得：
$$x_1\mathbf{a}_1 + \cdots + x_n\mathbf{a}_n = \mathbf{0}$$
这正是 $A\mathbf{x} = \mathbf{0}$ 的非零解。

### 3. 约束条件视角
方程组 $A\mathbf{x} = \mathbf{0}$ 包含：
- $m$ 个方程（约束条件）
- $n$ 个未知数

由于 $n < m$（未知数少于方程），这是一个**超定系统**。

**关键洞察**：虽然约束条件多，但当 $\text{rank}(A) < n$ 时，约束不是全部独立的，存在"冗余"约束，从而允许非零解存在。

## 💡 重要说明

### 特例情况
如果 $\text{rank}(A) = n$（最大秩），则：
- $\dim(\text{Null}(A)) = 0$
- 只有零解 $\mathbf{x} = \mathbf{0}$

但题目条件 $\text{rank}(A) \leq n$ 包含了 $\text{rank}(A) < n$ 的情况，此时必有非零解。

### 几何解释
在 $n$ 维空间中，$m$ 个超平面（$m > n$）的交集：
- 如果所有超平面都通过原点（齐次性）
- 且超平面不是全部线性无关
- 则交集必然包含除原点外的其他点

## ✅ 总结

$$\text{rank}(A) \leq n < m \Rightarrow n - \text{rank}(A) \geq 1 \Rightarrow \dim(\text{Null}(A)) \geq 1$$

因此齐次方程组 $A\mathbf{x} = \mathbf{0}$ 必定存在非零解。

**直观理解**：在"狭窄"的 $n$ 维空间中有"太多"($m$) 个约束，且这些约束不是完全独立的，必然存在"空隙"让非零解得以存在。

- 满足$A^2=A$的矩阵（不是线性空间）
- 上三角矩阵（是）

### 易错点：
- 条件型集合（如$A^2=A$）通常不封闭

# 二、线性映射（线性变换）与矩阵表示
## 1.线性映射与子空间
- **定义：**$T(x+y)=T(x)+T(y)$ ；$T(\alpha x)=\alpha T(x)$
- **映射核心**:零空间N(T)，像空间R(T)。对矩阵A：R(A)即A的列空间。
	- ps：N(T)=R(T)是有可能的
- **秩-零度定理：**
$$dim N(T)+dimR(T)=dim V$$
- **方程视角：**
	- $Ax=0\Leftrightarrow x\in N(A)$
	- $Ax=b有解\Leftrightarrow b\in R(A)$
	- 若有解：$解集=x_0+N(A)$
- [[方法-解空间交的维数求法]]
## 2.映射的性质（单，满，双射）
- 单射：不同的输入对应不同的输出$\Leftrightarrow N(T)=0$
- 满射：目标集的每个元素都有原像，即对任意b，Ax=b有解$\Leftrightarrow R(T)=W$
- 双射：一个函数既是单射又是满射。
- 方阵：$$N(A)={0}\Leftrightarrow A可逆\Leftrightarrow rank(A)=n$$
### some考点
- 线性映射T不一定保持线性无关性。
- 若某个无关组的像仍无关，不能推出T是单射or满射。
- 单射要求零空间只有零向量，等价于任意无关组的像仍无关。（只对一个无关组无关，不能推出对V的所有无关组都成立。）
- 线性映射单射➡️终点空间维数>=起点空间维数
- [[方法-哪些矩阵运算是线性变换]]
- 在$P_{n}[t]$上微分变换在任意基下都不可对角化
## 3.基与坐标
### 基变换公式：
$B_{1},B_{2}$为旧基和新基，$x,y$为坐标，$A_{1},A_{2}$为矩阵,$P$为过渡矩阵
基：$B_{2}=B_{1}P$
坐标变换：$x=Py$，$y=P^{-1}x$
矩阵：$A_{2}=P^{-1}A_{1}P$
### 一些小点：
- 向量不依赖基✅
- 坐标依赖基✅
- 相似=同一算子在不同基下的表示
- 维数唯一✅
- 一组个数等于空间维数且线性无关的向量自动成为一组基✅
- 在同一有限维空间，同一域上：不同基下矩阵相似➡️特征多项式/特征值相同
## 4.矩阵表示与相似
### 联系
- 给定基后：线性映射$\Leftrightarrow 矩阵$（一一对应）
- 不同基下矩阵相似
### 相似矩阵
定义：$$B=P^{-1}AP$$
$$A \sim B \iff \lambda I - A \cong \lambda I - B$$
- 即：若A和B相似，则它们的特征矩阵等价，初等因子和不变因子完全相同
### 相似不变量：
- 秩rank
- 行列式det
- 迹tr
- 特征值（重数）
#### 相似矩阵的判定：
- 先看特征值，找出几何重数，即找出Jordan块个数
- 然后对比选项的特征值，Jordan块个数
# 三、特征本质与Jordan标准形
## 1.特征值与特征向量
- $Ax=\lambda x(x\neq 0)$;**不同**特征值对应的特征向量线性无关。
- $AB$和$BA$:非零特征值，特征多项式，代数重数均相同，但秩不一定相同。
- 几何重数与代数重数[[100 学习相关/111 研一课程/矩阵理论/考试用的一些/线性映射与矩阵/几何重数与代数重数]]
- 几何重数（特征向量个数=Jordan块数量）$\leq$代数重数（特征值总量=Jordan块阶数和）
- 特征多项式$\Rightarrow$所有特征值及其总量（重数）
- 最小多项式各因式的幂次$\Rightarrow$对应$\lambda_{i}$的最大Jordan块的阶数
- Cayley-Hamilton定理[[Cayley-Hamilton定理]]
- 特征多项式与最小多项式相等$\Rightarrow$对每个特征值，几何重数=1$\Rightarrow$r(A)=n-1[[伴随矩阵列空间维数]]$\Rightarrow$此时结构唯一确定
- **盖尔圆盘定理**
	- 定义： 特征值 $\lambda$ 必落在所有行圆盘 $D_i$ 的并集中：$$D_i = \{ z \in \mathbb{C} : |z - a_{ii}| \le \sum_{j \ne i} |a_{ij}| \}$$
	- 转置对称性：由于 $A$ 与 $A^T$ 特征值相同，特征值也必落在所有**列圆盘**的并集中（通常取行、列交集来缩小范围）。
	- 孤立圆盘性质： 若 $k$ 个圆盘构成的连通分量与其他圆盘不相交，则该连通分量内恰好有 $k$ 个特征值。
	- **核心应用：**
	    - 判定可逆：若所有圆盘都不包含原点 $\Rightarrow$ 矩阵严格对角占优 $\Rightarrow$ 矩阵必可逆（$\det A \neq 0$）。
	    - 判定稳定性：若所有圆盘都在复平面左半部分 $\Rightarrow$ 特征值实部全为负 $\Rightarrow$ 系统稳定。
	    - 判定对角化：若所有圆盘互不相交 $\Rightarrow$ 特征值互异 $\Rightarrow$ 矩阵必可对角化。
### some考点
- 属于特征值$\lambda$ 的全部特征向量构成一个线性子空间。（❌因为不包括零向量）
- 任意特征值：几何重数$\leq$代数重数
- AB和BA具有相同的非零特征值，相同的特征多项式，相同的代数重数，但秩不一定相同。
- $x^n-1=0$在复数域上有且仅有n个不同的根
- 对于任意一个 $2 \times 2$ 的矩阵 $A$，其特征多项式 $f(\lambda) = \det(\lambda I - A)$ 有一个非常经典的简化展开形式：$f(\lambda) = \lambda^2 - \text{Tr}(A)\lambda + \det(A)$
- 矩阵幂运算的特征值是原特征值的幂，但代数重数不变。
## 2.判定（可对角化，可逆）
### 可对角化判据（单纯矩阵）
- 特征值互异$\Rightarrow$可对角化
- 几何重数=代数重数$\Leftrightarrow$可对角化
- 最小多项式无重根$\Leftrightarrow$可对角化$\Leftrightarrow$其Jordan块全是1阶的
### 可逆判据（不可逆（奇异矩阵））
$A 可逆 ⇔ ∃ A^{-1} 使 AA^{-1}=A^{-1}A=I$
以下条件等价：
- $det(A) ≠ 0$（$\lambda$矩阵要求行列式为非零**常数**，不能含$\lambda$)
- $rank(A) = n$
- A 的列向量（行向量）线性无关
- $Ax = 0$只有零解
- A 的最小多项式常数项 ≠ 0
- A 可以写成初等矩阵乘积
- 线性变换 $T$ 可逆

## 3.正定矩阵&半正定矩阵
- 正定$\Leftrightarrow$所有特征值>0，所有顺序主子式>0
- 半正定矩阵必须是所有主子式>0
- 实对称正定矩阵$\Leftrightarrow$存在$B$，使$A=B^2$

# 六、矩阵范数及级数
## 1. 范数 (Norms)
- **向量范数：** $L_1$（模和）、$L_2$（欧氏）、$L_\infty$（最大值）。
- 三大定理：
	1.非负性，对任意向量x，恒有$||x ||\geq 0$
	2.齐次性：对于任意常数（标量） $\alpha$ 和向量 $x$，有：$\|\alpha x\| = |\alpha| \cdot \|x\|$。
	3.三角不等式：对于任意向量 $x, y$，有：$\|x + y\| \le \|x\| + \|y\|$。
- **矩阵范数：**[[方法-矩阵范数计算]]
    **列和范数**
	    $||A||_1 = \max_{1 \le j \le n} \sum_{i=1}^m ||a_{ij}||$；计算每一列绝对值之和，取最大值
	**行和范数**
		$||A||_\infty = \max_{1 \le i \le m} \sum_{j=1}^n |a_{ij}|$；计算每一行绝对值之和，取最大值
	**谱范数**
		$||A||_2 = \sqrt{\lambda_{\max}(A^* A)} = \sigma_{\max}(A)$；
		计算 $B = A^* A$（实矩阵用 $A^T A$），求 $B$ 的最大特征值 $\lambda_{\max}$，然后开平方
	**Frobenius范数**
		$||A||_F = \sqrt{\sum_{i=1}^m \sum_{j=1}^n |a_{ij}|^2} = \sqrt{\operatorname{tr}(A^* A)}=\sqrt{\sum\sigma_{i}^2}$
		计算所有元素之和，然后开平方
		**Schur 不等式**：$\|A\|_F^2 \ge \sum_{i=1}^n |\lambda_i|^2$（正规矩阵时取等号）
- **相容性：**
	- $||Ax||_a \le ||A||_b ||x||_a$
		- $\|\cdot\|_a$ 是 $\mathbb{C}^n$ 上的向量范数，$\|\cdot\|_b$ 是 $\mathbb{C}^{n \times n}$ 上的矩阵范数。
- **重要不等式**
	- $\|x\|_\infty \le \|x\|_1 \le n \|x\|_\infty$
	- $\|x\|_1 = \sum |x_i|$，而 $\|x\|_\infty = \max |x_i|$。
	- $\|x\|_\infty \le \|x\|_2 \le \|x\|_1$
	- 固定成立的不等式：$||I-(I+X)^{-1}||\leq \frac{||X||}{1-||X||}$
	- 三角不等式$||A+B||\leq||A||+||B||$
	- 范数次乘性：$||AB||\leq||A||·||B||$
## 2. 矩阵级数与收敛
- **收敛判定：**
	- $\lim_{k \to \infty} A^k = 0 \iff \rho(A) < 1$。
	- 矩阵序列 $A^k$ 收敛（即 $\lim_{k \to \infty} A^k$ 存在）的充要条件是：
		1. 谱半径条件：所有特征值 $\lambda$ 满足 $|\lambda| < 1$；或者 $\lambda = 1$。
		2. Jordan 块条件：如果存在特征值 $\lambda = 1$，则该特征值对应的 Jordan 块必须全部为 1 阶（即其代数重数等于几何重数）。
- **谱半径 $\rho(A)$：** 
	- $\rho(A)=max\{|\lambda_{i}|\}$
	- 特征值绝对值的最大值。
- **Neumann 级数：** 
	- 若 $||A|| < 1$，则 $(I-A)^{-1} = \sum A^k$。
	- $(I+X)^{-1}=\sum_{k=0}^{\infty}(-1)^kX^k$
- **矩阵指数 $e^A$：**
	- $\det(e^A) = e^{\text{tr}(A)}$；特征值为 $e^{\lambda_i}$。
### 4.幂收敛矩阵的性质
- 当 $|x| < 1$ 时，几何级数的求和公式为：$\sum_{k=0}^{\infty} x^k = \frac{1}{1-x}$
- [[方法-矩阵幂级数的收敛判定]]
- 如果标量幂级数在收敛圆周 $|z|=R$ 上绝对收敛（如 $\sum \frac{1}{k^2}$），那么矩阵幂级数在 $\rho(A)=R$ 时也**绝对收敛**，且与其 Jordan 结构无关。
### 5.对数级数的收敛性：
- $f(A) = \sum_{k=1}^\infty \frac{A^k}{k}$是$-\ln(I-A)$ 的泰勒展开。
	- 收敛的充要条件是 $A$ 的所有特征值 $|\lambda_i| \le 1$ 且 $\lambda_i \neq 1$。

### 6.必背级数
$\sin A = A - \frac{A^3}{3!} + \frac{A^5}{5!} - \dots = \sum_{k=0}^{\infty} \frac{(-1)^k A^{2k+1}}{(2k+1)!}$
$\cos A = I - \frac{A^2}{2!} + \frac{A^4}{4!} - \dots = \sum_{k=0}^{\infty} \frac{(-1)^k A^{2k}}{(2k)!}$
$e^A = I + A + \frac{A^2}{2!} + \frac{A^3}{3!} + \dots = \sum_{k=0}^{\infty} \frac{A^k}{k!}$
$\frac{1}{1-x} = \sum x^k\Leftrightarrow(I-A)^{-1} = \sum A^k$   (收敛条件：$\rho(A) < 1$)
$\frac{1}{(1-x)^2} = \sum kx^{k-1}\Leftrightarrow(I-A)^{-2} = \sum kA^{k-1}$  (收敛条件：$\rho(A) < 1$)
$-\ln(1-x) = \sum \frac{x^k}{k}\Leftrightarrow-\ln(I-A) = \sum \frac{A^k}{k}$  (收敛条件$\rho(A) < 1$)
$\frac{1-x^n}{1-x} = \sum_{0}^{n-1} x^k\Leftrightarrow(I-A^n)(I-A)^{-1} = \sum_{0}^{n-1} A^k$(收敛条件：只要 $I-A$ 可逆)
### 3. 矩阵微积分
- **迹导数公式：**
    - $\frac{d}{dA} \text{tr}(BA) = B^T$
    - $\frac{d}{dA} \text{tr}(A^T B A) = (B+B^T)A$
- **矩阵函数的谱映射定理**：
	对于 $n$ 阶方阵 $A$ 和函数 $f(\lambda)$，如果 $\lambda_i$ 是 $A$ 的特征值，那么 $f(\lambda_i)$ 就是矩阵函数 $f(A)$ 的特征值。
- **计算矩阵函数通用法：（谱上一致法）**
	- 若 $A = P J P^{-1}$，则 $f(A) = P f(J) P^{-1}$
		- 若$J=\lambda I+N$，则$f(J)=f(\lambda)I+f'(\lambda)N$
- 考虑$A+B可逆时常用技巧：A+B=A(I+A^{-1}B)$
	- 常用结论:$||(I+X)^{-1}||\leq\frac{{1}}{1-||X||}$

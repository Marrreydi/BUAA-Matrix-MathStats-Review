### **一、 连续型随机变量**
1. **正态分布 $N(\mu, \sigma^2)$**(超级无敌爆炸重要的重中之重，背不下来就别去参加考试了。)
	第一个参数$\mu$：代表均值（期望值），即$E(X)=\mu$，决定了钟形曲线中心在横轴上的位置。
	第二个参数$\sigma^2$：代表方差，即$Var(X)=\sigma^2$，决定了曲线的胖瘦程度。
    - **概率密度**：$$f(x; \mu, \sigma^2) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left( -\frac{(x-\mu)^2}{2\sigma^2} \right), -\infty < x < \infty$$
    - $E(\bar{X})=\mu$
    - $Var(\bar{X})=\frac{\sigma^2}{n}=\frac{1}{n}$

2. **均匀分布 $U(a, b)$**
    - **概率密度**：$$f(x) = \frac{1}{b-a}, a \le x \le b$$
    - **常见变体**：真题中出现了 $U(\theta, 2\theta)$，其充分统计量为 $(X_{(1)}, X_{(n)})$ 。

3. **指数分布 $Exp(\theta)$**
    - **概率密度**：常见形式为 $$f(x; \theta) = \frac{1}{\theta} e^{-x/\theta}, x > 0$$（或 $\lambda e^{-\lambda x}$）。
    - 性质：$E(X_{i})=\theta,Var(X_{i})=\theta^2$

4. **伽马分布 $\Gamma(\alpha, \beta)$**
	ps：大概率不考，但我觉得对于UMVUE 的一些题目的求解过程的理解很重要。但是其实就是不考但是其实我就是想记所以别管了就这样吧。
	
	$Exp(\theta)$是$\Gamma(\alpha,\beta)$,当$\alpha=1,\beta=\theta$时，伽马分布就退化成了指数分布。
	指数分布描述的是**发生一次**随机事件等待的时间，而伽马分布 $\Gamma(n, \theta)$ 描述的就是发生 $n$ 次同样的随机事件总共需要等待的时间。
	- **概率分布**：对于$T \sim \Gamma(n,\theta)$：$$f_{T}(t)=\frac{\theta^{n}}{\Gamma(n)}t^{n-1}e^{-\theta t},t>0$$
	- $\Gamma(n)$：对于整数n，$\Gamma(n)=(n-1)!$
	- $E[1/T] = \frac{\theta}{n-1}$
    - $E[1/T^2] = \frac{\theta^2}{(n-1)(n-2)}$

### **二、 离散型随机变量**

1. **泊松分布 $P(\lambda)$**
    - **分布律**：$P(X=x) = \frac{\lambda^x e^{-\lambda}}{x!}, x = 0, 1, 2, \dots$ 。
    - 期望和方差都为$\lambda$
2. **二项分布 $B(m, \theta)$**
    - **分布律**：$P(X=x) = C_m^x \theta^x (1-\theta)^{m-x}, x = 0, 1, \dots, m$ 。
    - 期望是$m\theta$，方差是$m\theta(1-\theta)$，所以对于$\sum x_{i}$，期望为$nm\theta$
3. **几何分布**
    - **分布律**：$P(X=x) = \theta(1-\theta)^x, x = 0, 1, 2, \dots$ 。
    - **真题考点**：去年真题的贝叶斯估计填空题采用了此分布。

### **三、 三大抽样分布（填空题）**
这些分布通常用于构造检验统计量，需熟记其构造形式：

1. **卡方分布 $\chi^2(n)$**
    - **构造**：若 $X_i \sim N(0, 1)$ 且独立，则 $\sum_{i=1}^n X_i^2 \sim \chi^2(n)$。
    - **性质**：$E(\chi^2) = n, Var(\chi^2) = 2n$。
2. **$t$ 分布**
    - **构造**：$T = \frac{X}{\sqrt{Y/n}}$，其中 $X \sim N(0, 1), Y \sim \chi^2(n)$ 且独立。
3. **$F$ 分布**
    - **构造**：$F = \frac{X/n_1}{Y/n_2}$，其中 $X \sim \chi^2(n_1), Y \sim \chi^2(n_2)$ 且独立。


### 1.底层原理：什么是分布的变换。
$Y=AX+B$，本质上是在做坐标变换。
- 如果$X=(X_{1},X_{{2}})^T$服从线性分布，那么它们的线性组合$Y_{1},Y_{2}$依然服从正态分布。
	- 一个正态分布的特征完全由它的均值和方差决定。
- 对正态分布，独立等价于不相关。
	- 只要让$Cov(Y_{1},Y_{2}=0)$，它们就独立。


### 2.解题步骤
假设题目给出的已知条件是 $X \sim N(\mu, \Sigma)$，其中 $\mu$ 是均值向量，$\Sigma$ 是协方差矩阵。
$X \sim N_3(\mu, \Sigma)$中：
- $X$表示一个随机向量，因为下标是3，所以X是一个包含三个分量的列表：$X=[X_{1},X_{2},X_{3}]^T$
- $\sim$:读作服从于……的分布
- N：代表正态分布
- 3：表示纬度
- $\mu$:均值向量，表示三个变量各自的平均值
- $\Sigma$（协方差矩阵）：告诉你这三个变量各自的波动程度（方差），以及它们彼此之间的相关程度（协方差）。
#### step1：求Y的分布（利用期望和方差的线性性质）
**求均值 $E(Y)$：**
- $E(Y_1) = E(aX_1 + X_2) = aE(X_1) + E(X_2)$ 
- $E(Y_2) = E(X_1 + bX_2) = E(X_1) + bE(X_2)$
由此得到 $E(Y) = (E(Y_1), E(Y_2))^T$。
**求方差 $D(Y)$（协方差矩阵）**：
$D(Y_1)$，$D(Y_2)$ 和 $Cov(Y_1, Y_2)$。
- $D(Y_1) = Cov(aX_1 + X_2, aX_1 + X_2) = a^2 D(X_1) + D(X_2) + 2a Cov(X_1, X_2)$
- $D(Y_2) = Cov(X_1 + bX_2, X_1 + bX_2) = D(X_1) + b^2 D(X_2) + 2b Cov(X_1, X_2)$
- $Cov(Y_1, Y_2) = Cov(aX_1 + X_2, X_1 + bX_2)$
	- $Cov(X, Y) = E(XY) - E(X)E(Y)$
	- $D(\sum a_i X_i) = \sum a_i^2 D(X_i) + 2 \sum_{i<j} a_i a_j Cov(X_i, X_j)$
有一个结论一定要记住：
$\boxed{E(Y)=A\mu}$
$\boxed{Cov(Y)=A\Sigma A^T}$
	一维情况下：$Y=c^TX$,$Var(Z)=c^T\Sigma c$
#### step2：求常数a,b使其独立
列出一个方程：
$$Cov(Y_1, Y_2) = 0$$
$Cov(A+B, C+D) = Cov(A,C) + Cov(A,D) + Cov(B,C) + Cov(B,D)$
利用协方差的双线性性质（像多项式乘法一样展开）：
$$Cov(aX_1 + X_2, X_1 + bX_2) = a Cov(X_1, X_1) + ab Cov(X_1, X_2) + Cov(X_2, X_1) + b Cov(X_2, X_2)$$

$$= a D(X_1) + (ab+1) Cov(X_1, X_2) + b D(X_2)$$

令上述式子等于 $0$，解出 $a$ 和 $b$ 的关系。


### 二维情况
**step1:**
把题目改写成矩阵形式，即$Y=(Y_{1},Y_{2})^T=AX$
	$Y_{1}=c_{1}^T,Y_{2}=c_{2}^T$
**step2：**
因为$X\sim N_{3}(\mu,\Sigma)$，且Y=AX是线性变换，所以$\boxed{Y \sim N_{2}(\mu_{Y},\Sigma_{Y})}$
其中$\boxed{\mu_{Y}=A\mu;\Sigma_{Y}=A\Sigma A^T}$
**step3:算均值向量**
$E(Y_{1})=c_{1}^T\mu;E(Y_{2})=c_{2}^T\mu$
算完写成列向量$\mu_{Y}=(E(Y_{1}),E(Y_{2}))^T$
**step4:算协方差矩阵**
$$\Sigma_Y = \begin{pmatrix}
\operatorname{Var}(Y_1) & \operatorname{Cov}(Y_1, Y_2) \\
\operatorname{Cov}(Y_2, Y_1) & \operatorname{Var}(Y_2)
\end{pmatrix}$$
$Var(Y_{1})=c_{1}^T\Sigma c_{1};Var(Y_{2})=c_{2}^T\Sigma c_{2};Cov(Y_{1},Y_{2})=c_{1}^T\Sigma c_{2}=Cov(Y_{2},Y_{1})$
最后整合：$$\mathbf{Y} \sim N_2 \left(
\begin{pmatrix}
E(Y_1) \\
E(Y_2)
\end{pmatrix},
\begin{pmatrix}
\operatorname{Var}(Y_1) & \operatorname{Cov}(Y_1, Y_2) \\
\operatorname{Cov}(Y_1, Y_2) & \operatorname{Var}(Y_2)
\end{pmatrix}
\right)$$

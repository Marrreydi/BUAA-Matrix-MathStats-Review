**测均值找T、Z，测方差找卡方。**

在似然比检验（LRT）中，化简后的似然比统计量 $\lambda$ 通常是某个**单调函数**，我们要把它转化回我们熟悉的分布。

### 读题：
(1)判断是哪侧检验：
	如果是右侧检验，即$H_{1}:\sigma^2>\sigma_{0}^2;H_{1}:\mu>\mu_{0}$，则常用分布量的下标要用$\chi_{(1-\alpha)}^2,z_{1-\alpha},t_{1-\alpha}$,势函数用$1-F( )$
	如果是左侧检验，则用$\chi_{\alpha}^2,z_{\alpha}$等，势函数用$F$
(2)判断该用哪种常用分布：

| **总体分布**                  | **检验对象**      | **原假设 $H_{0}$​**        | **核心统计量构造**                                                | **$H_{0}$​ 下的分布** |
| ------------------------- | ------------- | ----------------------- | ---------------------------------------------------------- | ----------------- |
| **正态** $N(\mu, \sigma^2)$ | 均值 $\mu$      | $\mu = \mu_0$           | $Z = \frac{\bar{X}-\mu_0}{\sigma/\sqrt{n}}$ ($\sigma^2$已知) | $N(0,1)$          |
| **正态** $N(\mu, \sigma^2)$ | 均值 $\mu$      | $\mu = \mu_0$           | $t = \frac{\bar{X}-\mu_0}{S/\sqrt{n}}$ ($\sigma^2$未知)      | $t(n-1)$          |
| **正态** $N(\mu, \sigma^2)$ | 方差 $\sigma^2$ | $\sigma^2 = \sigma_0^2$ | $\chi^2 = \frac{\sum(X_i-\mu)^2}{\sigma_0^2}$ ($\mu$已知)    | $\chi^2(n)$       |
| **正态** $N(\mu, \sigma^2)$ | 方差 $\sigma^2$ | $\sigma^2 = \sigma_0^2$ | $\chi^2 = \frac{(n-1)S^2}{\sigma_0^2}$ ($\mu$未知)           | $\chi^2(n-1)$     |
| **指数** $Exp(\lambda)$     | 均值 $\lambda$  | $\lambda = \lambda_0$   | $\chi^2 = 2\lambda_{0}\sum X_i$                            | $\chi^2(2n)$      |
| **泊松** $P(\lambda)$       | 强度 $\lambda$  | $\lambda = \lambda_0$   | $W = \sum X_i$                                             | $P(n\lambda_0)$   |
| **二项** $B(m, p)$          | 成功率 $p$       | $p = p_0$               | $Y = \sum X_i$                                             | $B(nm, p_0)$      |

### 2. 写出拒绝域的具体形式 🛡️
拒绝域的逻辑永远是：当 $\lambda$ 越小时，说明原假设 $H_0$ 越不可信，越倾向于拒绝 $H_0$。
所以拒绝域的形式总是：$R = \{ \lambda < C \}$。
由于 $\lambda$ 通常是关于某个统计量 $W$ 的函数 $g(W)$，我们需要看 $g(W)$ 的单调性：
- **单调递增**：$\{ \lambda < C \}$ 对应 $\{ W < C' \}$
- **单调递减**：$\{ \lambda < C \}$ 对应 $\{ W > C' \}$

---
### 3. 大样本的“救命稻草”：威尔克斯定理 (Wilks' Theorem) 🎓
如果你实在凑不出精确分布（比如总体不是正态分布），且样本量 $n$ 很大，可以使用这个万能公式：
$$-2 \ln \lambda \xrightarrow{d} \chi^2(k)$$
其中 $k$ 是参数空间的维数差（通常是 $H_1$ 下独立参数个数减去 $H_0$ 下独立参数个数）。

### 例题：(24-25考题)
设$x_{1}\dots x_{n}$是来自正态总体$N(0,\sigma^2)$的简单样本，在显著性水平$\alpha$下：
1.给出下面假设检验问题的似然比检验。
2.给出该似然比检验的势函数$g(\sigma^2)$;
3.判断该检验是否为无偏检验。
$H_{0}:\sigma^2\leq \sigma_{0}^2;H_{1}:\sigma^2>\sigma_{0}^2$

答：1.读题可知：此题检验$\sigma^2$，其中$\mu$已知，所以要用的核心统计量为：$\frac{\sum (x_{i}-\mu)^2}{\sigma_{0}^2}$
且此题为右侧检验，所以下标用$1-\alpha$，势函数用1-F(·)（ps：韦卫老师说用下分位哦）
即最终要找到一个$\frac{\sum (x_{i}-\mu)^2}{\sigma_{0}^2}\sim \chi_{(1-\alpha)}^2(n)$
(1)构建似然函数
$L(\sigma^2)=(2\pi \sigma^2)^{-n/2}\exp\left[ -\frac{\sum(x_{i})^2}{2\sigma^2} \right]$
(2)构建似然比：
分母：找全空间的最大似然比
即$\hat{\sigma^2}=\frac{1}{n}\sum x_{i}^2$，$sup_{H}L(\hat{\sigma^2})=(2\pi\hat{\sigma^2})^{-n/2}\exp\left( -\frac{n}{2} \right)$
分子：找约束下的最大似然比，即$\hat{\sigma_{0}}^2$,$sup_{H_{0}}L=(2\pi \sigma_{0}^2)^{-n/2}\exp\left( -\sum \frac{x_{i}^2}{2\sigma_{0}^2} \right)$
构建似然比：$$\Lambda=\frac{sup_{H_{0}}L}{sup_{H}L}=\left( \frac{\hat{\sigma}^2}{\sigma_{0^2}} \right)^{n/2}\exp\left( -\frac{\sum x_{i}^2}{2\sigma_{0}^2} +\frac{n}{2} \right)$$
设$Q=\sum \frac{x_{i}^2}{\sigma_{0}^2}$,则$\left( \frac{\hat{\sigma}^2}{\sigma_{0^2}} \right)=\frac{Q}{n}$似然比化简为：$$\Lambda=\left( \frac{Q}{n} \right)^{n/2}\exp\left( \frac{n-Q}{2} \right)$$
又因为当$\Lambda$不等于 1 的时候，是处在 H1 范围内，即$\hat{\sigma}^2>\sigma_{0}^2;即Q>n$，所以随着统计量 Q 的增加，$\Lambda$减小。
所以拒绝域$\Lambda\leq C \Leftrightarrow Q\geq C'$
又因为$Q=\frac{\sum x_{i}^2}{\sigma_{0}^2}\sim \chi^2(n)$
所以拒绝域为$W=\{\frac{\sum x_{i}^2}{\sigma_{0}^2}\geq\chi_{1-\alpha}^2(n)\}$
2.
势函数为$g(\sigma^2)=P\left\{ \frac{\sum x_{i}^2}{\sigma_{0}^2}\geq\chi_{1-\alpha}^2(n) \right\}=P\left\{ \frac{\sum x_{i}^2}{\sigma^2}\cdot \frac{\sigma^2}{\sigma_{0}^2}\geq\chi_{1-\alpha}^2(n) \right\}=P\left\{ \frac{\sum x_{i}^2}{\sigma^2}\geq\frac{\sigma_{0}^2}{\sigma^2}\chi_{1-\alpha}^2(n) \right\}=1-F_{\chi^2}(\frac{\sigma_{0}^2}{\sigma^2}\chi_{1-\alpha}^2(n))$
3.无偏检验

- 令 $h(\sigma^2) = \frac{\sigma_0^2}{\sigma^2} \chi_{1-\alpha}^2(n)$。由于 $\chi_{1-\alpha}^2(n) > 0$，当 $\sigma^2$ 增大时，$h(\sigma^2)$ 是严格递减的。
- 卡方分布的累积分布函数 $F_{\chi^2(n)}(y)$ 是关于 $y$ 的单调递增函数，因此 $F_{\chi^2(n)}(h(\sigma^2))$ 随着 $\sigma^2$ 的增大而**减小**。
- 进而，$g(\sigma^2) = 1 - F_{\chi^2(n)}(h(\sigma^2))$ 随着 $\sigma^2$ 的增大而**严格递增**。
$g(\sigma_{0}^2)=P\left\{ \frac{\sum x_{i}^2}{\sigma_{0}^2}\geq\frac{\sigma_{0}^2}{\sigma_{0}^2}\chi_{1-\alpha}^2(n) \right\}=\alpha$
当 $\sigma^2 > \sigma_0^2$ 时，恒有 $g(\sigma^2) > \alpha$，同时，对于 $\sigma^2 \leq \sigma_0^2$（即 $H_0$ 范围内），有 $g(\sigma^2) \leq \alpha$。
所以是无偏检验。
